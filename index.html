Sorting Algorithms II: Quicksort

Link to source code

Quicksort is a divide and conquer algorithm with the same goals as bubble sort as to organize the items inside the list. The algorithm partitions the list into three sections by making recursive calls. Recursion is when a function gets called from within itself.

Sorting Algorithms III: Merge Sort

Link to source code

Like quicksort, merge sort is another divide and conquer algorithm. Merge sort divides the list into multiple sub-lists until each sublist is a single element. The sub-lists get merged back, which results in a sorted list. The algorithm is one of the most efficient algorithms, with a runtime of O(n * logn).

Stack Data Structure Visualization

Link to source code

This program visualizes the behavior of the stack. A stack is a data structure that contains a collection of items with two primary operations: push and pop. Push adds an element to the top (or end) of the list (when the stack is not full). Pop removes the item from the same area (when it is not empty). Therefore, the following two mnemonics best describe stacks: FILO (First In, Last Out) and LIFO (Last In, First Out).

Queue Data Structure Visualization

Link to source code

This program is an interactive visualization of the queue, similar to the stack program. A queue is a data structure that (like the stack) contains a collection of items and has two operations: enqueue and dequeue. Enqueue adds elements to the list sequentially, while dequeue removes the front-most item in the group. The queue's associative mnemonic is FIFO (First In, First Out). A real-world example of a queue is the check-out at a grocery store.

Euler's Totient (phi) Function

Link to source code

Also known as Euler's phi (ϕ) function, Euler's totient function counts the positive integers of a given number that are relatively prime to that number. This program calculates the phi function of the first 50 integers. Each circle displayed graphically is an integer that does not share common factors with the number. For example, consider the prime number 47. Since 47 does not share a factor with any number, there is a vertical line of circles. Thus, ϕ(47) = 46. In addition to the graph, a drop-down is available to view the phi function for the first 50 integers.

Pacer Exercise Data Visualization

Link to source code

This 200-line object-oriented program visualizes the exercise data that I collected from the Pacer app. The data collection spans over two and a half years. The program utilizes file output to grab the data from a CSV file. Bubble size is pre-determined by the number of steps for a given day. The frame rate can change via a slider, as well as a reset button can restart the animation. Over 1000 objects instantiated.
