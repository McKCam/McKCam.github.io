Pacer Exercise Data Visualization

Link to source code

This object-oriented program visualizes the exercise data that I collected from the Pacer app. The data collection spans over two and a half years. The program utilizes file output to grab the data from a CSV file. Bubble size is pre-determined by the number of steps for a given day. Under the bubbles section is an animated bar chart, which displays the bar at the same as its corresponding bubble. At the end of the animation, a summary gets printed. The frame rate can change via a slider, as well as a reset button can restart the animation. Over 1000 objects instantiated.

Numerical Integration Visualized

This program calculates the area (or the Riemann sum) under the curve of a chosen mathematical function with given limits. The function's graph and the rectangles between the limits of integration get drawn to a canvas. There are 16 different functions to choose from with a sliding scale on the lower left to zoom in and out of the graph.

Queue Data Structure Visualization

Link to source code

This program is an interactive visualization of the queue, similar to the stack program. A queue is a data structure that (like the stack) contains a collection of items and has two operations: enqueue and dequeue. Enqueue adds elements to the list sequentially, while dequeue removes the front-most item in the group. The queue's associative mnemonic is FIFO (First In, First Out). A real-world example of a queue is the check-out at a grocery store.

Stack Data Structure Visualization

Link to source code

This program visualizes the behavior of the stack. A stack is a data structure that contains a collection of items with two primary operations: push and pop. Push adds an element to the top (or end) of the list (when the stack is not full). Pop removes the item from the same area (when it is not empty). Therefore, the following two mnemonics best describe stacks: FILO (First In, Last Out) and LIFO (Last In, First Out).
