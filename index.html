Numerical Integration (Visualized)

Link to source code

This 325-line program calculates the area (or the Riemann sum) under the curve of a chosen mathematical function with given limits. The function's graph and the rectangles between the limits of integration get drawn to a canvas. There are 16 different functions to choose from with a sliding scale to zoom in and out of the graph.

Numerical Integration (Animated)

Link to source code

Similar to the previous program, this project animates the rectangles forming (and transforming) under the curve. The area also updates as shapes get added.
