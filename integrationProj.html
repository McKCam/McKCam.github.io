Project Description:

The updated single-variable numerical integration visualizer now allows for *responsible* users to input their own math functions. The users can enter math functions using x as their variable. The function parser is imported from the math.js library and is encapsulated in a MathFunc class. This has simplified the code in the original sketch file from around 360 lines to 240 lines. The result is stored in the lower left input box.
